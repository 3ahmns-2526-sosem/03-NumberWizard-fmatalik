# **Number Wizard Game**



### **Beschreibung**

### 

### Number Wizard ist ein einfaches Ratespiel, bei dem der Spieler an eine Zahl denkt und der Computer versucht, diese zu erraten. Der Computer verwendet dabei die Methode der binären Suche, um die richtige Zahl möglichst schnell zu finden.

### 

### **Spielziel**

### 

### Denke dir eine Zahl innerhalb des vorgegebenen Zahlenbereichs. Der Computer stellt Vermutungen an, und du gibst ihm Hinweise, bis die richtige Zahl gefunden wurde.

### 

### ***Steuerung***

### **Higher**

### 

### Klicke auf Higher, wenn deine gedachte Zahl größer ist als der aktuelle Tipp des Computers.

### 

### **Lower**

### 

### Klicke auf Lower, wenn deine gedachte Zahl kleiner ist als der aktuelle Tipp des Computers.

### 

### **Correct**

### 

### Klicke auf Correct, wenn der Computer die richtige Zahl erraten hat.

### 

### **Restart**

### 

### Klicke auf Restart, um das Spiel neu zu starten und eine neue Zahl auszuwählen.

### 

### **Binäre Suche**

### 

### Das Spiel verwendet den Algorithmus der binären Suche. Dabei wird der mögliche Zahlenbereich nach jedem Hinweis halbiert. Dadurch kann der Computer die gesuchte Zahl sehr effizient finden und benötigt nur wenige Versuche.
